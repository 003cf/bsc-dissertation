Experimental Dashboard

Run your own adversarial experiment on a Convolutional Neural Network pre-trained on the MNIST dataset!

Hi there! Welcome ! Here you can test out some of the experiments I tried, once they're all run then you plot will pop up beneath this text. Quick run down: The models you're training are being trained on the MNIST dataset, within each training set theres a guaranteed 42,000 clean samples and 14,000 poisoned samples to implement a backdoor of your choosing. You can control how many adversarial samples you want to include using the slider over there :p The whole adversarial training dataset consists of 42,000 images, so any percentage you pick will be out of that.

We're all for repeatability over here, but also the training times do take forever so I don't really recommend choosing more than one epoch or experiment. But you do you !

'Epochs' refers to the number of training iterations the model will go through for the same dataset.

'Experiments' refers to the number of models that will be tested to draw an average. These experiments take time, so please be patient.
